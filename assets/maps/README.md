# Map Assets

This directory contains map images used by the application.

## Adding Maps

1. Place your map image file in this directory
2. Update the `defaultMapImagePath` in `src/utils/Constants.ts` to point to your image
3. Ensure the image is in a web-friendly format (PNG, JPG, WebP)
4. The image should have good resolution (recommended minimum 2048x2048)

## Current Maps

- `tellene-map.png` - The main world map for the Tellene campaign

## Notes

- Map images are NOT tracked by git (see `.gitignore`)
- Keep backup copies of your maps in a safe location
- High-resolution maps will provide better zoom experience
